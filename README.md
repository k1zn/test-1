# ЛР-5

RailwayTickets — это современная веб-платформа для поиска и покупки железнодорожных билетов. Система позволяет пользователям просматривать актуальное расписание поездов, выбирать доступные места на интерактивных схемах вагонов и проводить безопасную оплату. Администраторы платформы могут управлять маршрутами, добавлять новые составы и отслеживать заполняемость рейсов. Проект направлен на автоматизацию процесса продажи билетов и улучшение клиентского опыта. Основная цель — обеспечить высокую скорость обработки транзакций и удобный интерфейс для пассажиров.

## Sprint Retrospective

**Что прошло хорошо**  
- Удалось полностью закрыть все high‑приоритетные задачи спринта: регистрацию пользователей и выбор мест, что снижает риск срыва релиза по критическому функционалу.
- Планирование по объему high/medium задач оказалось реалистичным: самые важные пользовательские сценарии покрыты.

**Что пошло не так**  
- Оценка по задачам с интеграциями и инфраструктурой оказалась заниженной: поиск поездов и интеграция оплаты требуют больше времени из‑за внешних API и сложности бизнес‑логики.
- Параллельный старт нескольких задач (поиск поездов, генерация PDF‑билета) привёл к распылению фокуса, из‑за чего ни одна из них не была доведена до Done в рамках спринта.

**Что можно улучшить в следующем спринте**  
- Жёстче ограничивать WIP: не начинать новые задачи, пока текущие спринтовые истории не доведены до статуса Done, особенно для **интеграций** и генерации документов.
- Улучшить оценку и декомпозицию тяжёлых задач: разбивать интеграцию оплаты и поиск поездов на подзадачи (работа с API, валидация, UI, тесты), чтобы точнее планировать и видеть ранний прогресс.
